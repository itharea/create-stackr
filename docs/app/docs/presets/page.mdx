# Presets

Choose a preset to get started quickly with a pre-configured setup.
Each preset is optimized for different use cases, from minimal prototypes
to full production apps with analytics.

## Minimal

Start with just the essentials: authentication, session management, and tab navigation.
No integrations or onboarding flows - perfect for building from scratch.

<CodeBlock code="npx create-stackr@latest my-app --template minimal" language="bash" />

## Full-Featured

The complete package for production apps. Includes all features (authentication, onboarding
with 3 pages, paywall) and all integrations (RevenueCat, Adjust, Scate, ATT). Event queue
enabled for background processing.

<CodeBlock code="npx create-stackr@latest my-app --template full-featured" language="bash" />

## Analytics-Focused

Optimized for apps that prioritize attribution and analytics. Includes Adjust, Scate,
and ATT integration with a streamlined 2-page onboarding. No RevenueCat or paywall -
ideal for ad-supported or free apps.

<CodeBlock code="npx create-stackr@latest my-app --template analytics-focused" language="bash" />
